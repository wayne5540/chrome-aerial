# Apple TV Aerial - Chrome Extension

Play a Apple TV Aerial video in your chrome new tab.

Inspired by 

* [Benjamin Mayo](http://benjaminmayo.co.uk/watch-all-the-apple-tv-aerial-video-screensavers)
